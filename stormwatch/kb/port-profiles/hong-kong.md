# Port Profile: Hong Kong (HKHKG)

- Code: HKHKG
- Location: 22.29°N, 114.16°E
- Country: Hong Kong
- Weather Vulnerability Score: 0.70

## Key Facts
- 9th busiest container port globally (18M TEU/year)
- Located on northern edge of South China Sea typhoon belt
- Typhoon signal system provides graded operational restrictions
- Critical for Shenzhen–Manila and Hong Kong–Los Angeles routes

## Weather Risks
- Typhoon exposure: 6–8 typhoons within 300km annually
- Typhoon Signal 8+: Port operations fully suspended (T8 = sustained winds >63 km/h)
- Average 2–3 T8+ events per year, each causing 1–3 days closure
- Combined HK+Shenzhen closure amplifies impact: 40% of South China cargo capacity offline

## Mitigation Options
- Typhoon Signal 3 (T3): Partial operations continue, vessels can still depart
- Shenzhen alternate (CNSZX): Same weather system, same risk profile
- Shanghai alternate (CNSHA): Adds 3–4 days but provides typhoon-separated option
